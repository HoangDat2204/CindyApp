Tài Liệu Cấu Trúc: API File Service (OCR Extraction Pipeline)

File: backend/api/fileservice.py

1. Khai báo Router:

Tạo router = APIRouter(prefix="/files", tags=["File Service"]).

2. Viết các Hàm Giữ Chỗ (Mock/Placeholder Functions):

Viết 3 hàm xử lý logic tách biệt bên ngoài router (để tôi có thể tự điền code vào sau). Tham số nhận vào nên là nội dung file (bytes) hoặc file path.

  • def identify_file_owner(file_bytes: bytes) -> dict:
    Hãy để hàm này cho tôi tự định nghĩa
  • def process_supplier_file(file_bytes: bytes, supplier_id: int) -> dict:
    Hàm để tôi tự định nghĩa
  • def process_client_file(file_bytes: bytes, client_id: int) -> dict:
    Hàm này để cho tôi tự định nghĩa

3. Viết API Endpoint chính: POST /files/extract

Nhiệm vụ của Endpoint:

  • Nhận file upload từ Frontend bằng UploadFile = File(...) từ FastAPI.
  • Đọc nội dung file bằng await file.read().
  • Gọi hàm identify_file_owner() để lấy thông tin phân loại.
  • Dùng câu lệnh rẽ nhánh if/elif kiểm tra owner_type:
    • Nếu là "supplier": Gọi tiếp hàm process_supplier_file().
    • Nếu là "client": Gọi tiếp hàm process_client_file().
    • Nếu không xác định được: raise HTTPException(status_code=400, detail="Không thể nhận diện loại file").
  • Trả về kết quả JSON cuối cùng cho Frontend.

4. Yêu cầu Output code:

Trả về mã nguồn Python hoàn chỉnh cho file fileservice.py. Code cần sạch sẽ, có try/except đầy đủ để bắt lỗi khi đọc file, và nhắc người dùng nhớ include router này vào main.py.
